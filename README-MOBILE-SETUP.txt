# app-banking-main

This folder contains multiple pieces, but the real banking app lives here:

- FinancialApp-main/ -> main project
- FinancialApp-main/frontend/ -> actual Next.js frontend
- rontend_incomplete_backup/ -> older incomplete frontend experiment kept as backup

## What to run

Backend:
- open FinancialApp-main/
- run 
pm install
- run 
pm run start:dev

Frontend:
- open FinancialApp-main/frontend/
- run 
pm install
- run 
pm run dev

## Current mobile status

The frontend is a mobile-friendly web app / PWA, not a native Android/iPhone app yet.
See FinancialApp-main/MOBILE_STATUS.md for details.
